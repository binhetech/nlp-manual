# 简介
本仓库主要用于收集、整理公开的自然语言处理（NLP）相关的技术资料，包括最新文献、经典书籍以及行业研究报告等。一方面便于自己对自然语言处理进行系统的认知、学习，另一方面也希望可以分享、并帮助到其他爱学习知识的人士。

目录结构的说明如下：
+ books: 专业书籍，格式一般为pdf
+ literature: 最新文献，格式一般为pdf
+ report: 研究报告，格式一般为pdf
+ docs: 其他可公开的文档，格式一般为doc, docx
